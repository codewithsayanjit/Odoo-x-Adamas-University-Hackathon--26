# 🚀 Odoo x Adamas University Hackathon '26

## Smart HRMS Portal (Human Resource Management System)

A modern and responsive **Human Resource Management System (HRMS)** developed during the **Odoo x Adamas University Hackathon '26**.

The project aims to simplify employee management by providing a centralized platform for employee records, authentication, attendance tracking, payroll management, leave management, and administrative monitoring.

---

## 📌 Project Overview

Managing employee information manually can be time-consuming and error-prone.

This HRMS Portal provides an efficient digital solution that enables organizations to:

* Manage employee records
* Handle employee registration and login
* Monitor attendance
* Manage payroll information
* Track leave requests
* Access administrative dashboards
* Generate workforce insights
